# Integrations

## Supabase
- **Authentication**: Used for administrative access (email/password).
- **Database**: PostgreSQL via Supabase JS client.
- **Real-time**: Not explicitly mentioned but supported by the client.

## Vercel
- Configuration in `vercel.json` for frontend deployment.

## Browser APIs
- **LocalStorage**: Used for theme persistence (`apd-theme`).
- **Custom Events**: `themeChanged` event for cross-component theme synchronization.
