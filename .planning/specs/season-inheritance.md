# HERANÇA DE TEMPORADAS E PERFIS PÚBLICOS

Ao criar nova temporada de uma competição existente, o sistema deve herdar automaticamente toda configuração estrutural da competição base.

## Herdar:
* formato
* regras
* critérios desempate
* estrutura de fases
* campos de inscrição
* métricas estatísticas
* layout público
* configurações administrativas

## O admin deve escolher:
1. Nova temporada
2. Duplicar competição
3. Criar do zero

As estatísticas devem reiniciar por temporada.
O histórico deve permanecer acessível.

## PERFIS PÚBLICOS AUTOMÁTICOS
Cada equipe aprovada deve gerar automaticamente página pública com elenco completo.
Cada jogador aprovado deve aparecer publicamente vinculado à temporada correspondente.
